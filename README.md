# Spring Boot Use Case Projects  This repository contains multiple Spring Boot mini projects built to practice REST API development and backend architecture. 
## 1.CarParkingSystem
A REST API to manage car parking records using CRUD operations.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- REST APIs

 ## 2.HotelRoomBookingSystem
